# Autoattack-game
Проект, вдохновлённый Vampire Survivors, собственно ходишь бьёшь мобов прокачиваешься.  
Готово:
- один персонаж
- несколько скиллов: Orbit, Garlic, Lighting, Knife, MagicProjectile, Whip, Lighting
- прокачка
- один вид противников
